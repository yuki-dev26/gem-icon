# Gem Icon Changer

Gemini アプリ の Gem アイコンを好きな画像に変更できる Chrome 拡張機能です。

## 機能

- Gemini アプリの会話画面に表示される Gem アイコンをカスタマイズ
- お気に入りの画像を簡単にアップロード
- リアルタイムでプレビュー表示
- 画像は拡張機能内に保存され、ページをリロードしても維持

## インストール方法

1. このリポジトリをクローンまたはダウンロード

```bash
git clone
```

2. Chrome で `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」を ON にする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

## 使い方

1. 拡張機能アイコンをクリック

2. 「画像を選択...」ボタンから好きな画像を選択

3. プレビューで確認

4. 「保存して適用」をクリック

5. Gemini のページ（https://gemini.google.com/app）をリロード

6. Gem アイコンが選択した画像に変更されます！

## ファイル構成

```
gem-icon/
├── manifest.json      # 拡張機能の設定ファイル
├── popup.html         # ポップアップのHTML
├── popup.js           # ポップアップの動作
├── style.css          # ポップアップのスタイル
├── content.js         # Geminiページでのアイコン置き換え処理
├── app_icon.png       # 拡張機能のアイコン
└── README.md          # このファイル
```

## 注意事項

- 拡張機能を更新した場合は、Gemini のページをリロードしてください
- 円形に表示されるため、正方形の画像が最適です
- 全ての Gem に画像が適用されてしまいます
- 拡張を無効にしたり削除すれば全てリセットされます
- 非公式の拡張機能となりますので自己責任でお使いください
